## FAQ

1. 왜 Node.js 12.18.3 인가요?
   Programmers 의 Node.js 버젼은 12.18.3 이며, BOJ는 14.15.0 입니다. 두군데 모두를 아우를 수 있기 위해 더 낮은 Node.js 버젼인 12.18.3 기준으로 소스코드를 작성하였습니다.
